# Infinitesimal Limit Conversion

## R - 原文

> 函数具有极限 A 的充分必要条件是 f(x)=A+alpha，其中 alpha 是无穷小。
>
> - 同济大学数学系，第一章第四节

## I - 方法论骨架

无穷小是极限计算的误差语言。
如果 `f(x)` 趋于 `A`，就可以写成 `f(x)=A+alpha`，其中 `alpha->0`。
反过来，只要能把函数写成常数加无穷小，就证明了极限。
这让极限四则运算变成无穷小代数：无穷小的和、差、有限倍、与有界函数的乘积仍是无穷小。
遇到无穷大量时，也可通过倒数转成无穷小关系。

## A1 - 书中的应用

### 案例 1: 极限与无穷小的关系

- 问题: 如何证明 `lim f(x)=A`。
- 方法论的使用: 教材把它改写为 `f(x)-A` 是无穷小。
- 结论: 极限证明可转为误差项控制。
- 结果: 后续极限运算法则获得统一证明路径。

### 案例 2: 无穷大与无穷小的倒数关系

- 问题: 如何判断函数趋于无穷大。
- 方法论的使用: 教材说明在函数不为零时，`f` 为无穷大与 `1/f` 为无穷小互相关联。
- 结论: 无穷大量可通过倒数转化为无穷小。
- 结果: 把难处理的发散趋势转成趋零判断。

## A2 - 触发场景

### 用户会在什么情境下需要这个 skill?

1. 需要证明某表达式的极限但不想直接使用 epsilon-delta。
2. 题目让判断某个量是否为无穷小或无穷大。
3. 需要解释极限运算法则为什么成立。
4. 需要把 `f(x)-A` 作为误差项来处理。

### 语言信号

- “用无穷小证明”
- “为什么 f(x)=A+alpha”
- “这个是不是无穷小”
- “无穷大和无穷小有什么关系”
- “极限运算法则怎么来的”

### 与相邻 skill 的区分

- 与 `equivalent-infinitesimal-substitution` 的区别: 本 skill 处理无穷小的基础转换；后者处理比值极限中的等价替换。
- 与 `limit-definition-proof` 的区别: 本 skill 不展开 epsilon 量词证明。

## E - 可执行步骤

1. **确定变化过程**
   - 完成标准: 明确是 `x->x0`、`x->infty` 还是 `n->infty`。

2. **构造误差项**
   - 完成标准: 写出 `alpha=f(x)-A`，或把目标表达式拆成常数加误差。

3. **证明误差为无穷小**
   - 完成标准: 用已知极限、无穷小运算、有界乘无穷小或倒数关系证明 `alpha->0`。

4. **回到极限结论**
   - 完成标准: 明确写出“因此原函数极限为 A”或“因此该量为无穷大/无穷小”。

## B - 边界

### 不要在以下情况使用

- 用户需要的是等价无穷小表的替换计算。
- 表达式中出现未定式，需要更具体的化简策略。

### 失败模式

- 没说明变化过程，就谈无穷小。
- 把“很小”当成“无穷小”，缺少极限语境。
- 忽略倒数关系中函数不为零的条件。
