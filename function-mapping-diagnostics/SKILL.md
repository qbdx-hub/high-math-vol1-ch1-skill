# Function Mapping Diagnostics

## R - 原文

> 构成一个映射必须具备以下三个要素：集合 X，集合 Y，对应法则 f。
>
> - 同济大学数学系，第一章第一节

## I - 方法论骨架

函数题的第一步不是计算，而是确认对象是否合法。
一个函数由定义域和对应法则共同决定；映射还要区分值域和到达集合。
反函数要求原函数在所讨论定义域上单射。
复合函数要求内层函数的输出落入外层函数的定义域。
分段表达式可以表示同一个函数，但每段的适用区间必须覆盖且不冲突。
许多高数错误不是算错，而是在不合法对象上开始计算。

## A1 - 书中的应用

### 案例 1: 反正弦函数的主值

- 问题: 三角函数本身不是全局一一映射，不能直接全局求反函数。
- 方法论的使用: 教材把正弦函数限制到主值区间，使它成为单射，再定义 `arcsin x`。
- 结论: 反函数存在性来自“限制定义域后的一一性”。
- 结果: 反三角函数拥有明确的定义域和值域。

### 案例 2: `sqrt(tan x)` 的复合合法性

- 问题: `tan x` 的自然值域含负数，而平方根函数要求输入非负。
- 方法论的使用: 先检查内层值域是否落入外层定义域，再限制 `x` 的范围。
- 结论: 不能只看表达式形式说它是复合函数，必须给出合法定义域。
- 结果: 复合函数的定义域是内层定义域的子集。

## A2 - 触发场景

### 用户会在什么情境下需要这个 skill?

1. 题目要求求函数自然定义域、判断两个函数是否相同。
2. 题目给出 `f(g(x))`、`g(f(x))`，要求求复合函数或其定义域。
3. 题目问某函数是否存在反函数，或要求求反函数。
4. 分段函数、绝对值函数、符号函数、取整函数等对象需要画图或判断性质。

### 语言信号

- “这个函数的定义域怎么求”
- “这两个函数是不是同一个”
- “能不能构成复合函数”
- “这个函数有没有反函数”
- “这个分段式到底是不是一个函数”

### 与相邻 skill 的区分

- 与 `continuity-discontinuity-diagnostics` 的区别: 本 skill 先判断函数对象是否合法；后者判断某点连续或间断。
- 与 `limit-operation-and-substitution` 的区别: 本 skill 不计算极限，只给极限计算前的定义域和复合条件。

## E - 可执行步骤

1. **列出函数三要素**
   - 完成标准: 写出定义域、对应法则和值域范围；若是映射，还区分值域与到达集合。

2. **检查单值性和区间条件**
   - 完成标准: 对每个允许输入，确认只有一个输出；分段函数要检查分段边界是否冲突或遗漏。

3. **处理反函数或复合函数**
   - 完成标准: 反函数题检查单射；复合函数题检查内层值域是否落入外层定义域。

4. **输出合法对象**
   - 完成标准: 给出最终定义域、函数表达式和必要限制；若不合法，说明失败条件。

## B - 边界

### 不要在以下情况使用

- 用户已经给出合法函数并只要求求极限、导数或积分。
- 题目是纯集合论或抽象代数映射问题，超出第一章函数基础范围。

### 失败模式

- 把值域范围误当成实际值域。
- 忘记反函数需要单射。
- 写复合函数时只拼表达式，不检查外层定义域。

### 作者/教材边界

- 第一章只讨论一元实函数为主，复杂多元映射不在本 skill 覆盖范围内。
