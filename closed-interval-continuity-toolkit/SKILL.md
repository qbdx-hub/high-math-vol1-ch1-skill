# Closed Interval Continuity Toolkit

## R - 原文

> 在闭区间上连续的函数在该区间上有界且一定能取得最大值和最小值。
>
> - 同济大学数学系，第一章第十节

## I - 方法论骨架

闭区间连续性是一组存在性定理的总开关。
如果函数在 `[a,b]` 上连续，就能使用有界性与最值定理。
如果端点函数值异号，就能用零点定理证明方程至少有根。
如果要证明函数取到端点值之间的某个数，就用介值定理。
如果要把点态连续升级为统一控制，就用一致连续性定理。
每次调用前都必须先核查两个条件：闭区间、全区间连续。

## A1 - 书中的应用

### 案例 1: 三次方程有根

- 问题: 证明 `x^3-4x^2+1=0` 在 `(0,1)` 内至少有一个根。
- 方法论的使用: 教材验证函数在 `[0,1]` 上连续，且端点函数值异号。
- 结论: 根据零点定理，开区间内至少有一个零点。
- 结果: 方程存在根。

### 案例 2: 闭区间连续函数取最值

- 问题: 判断函数是否有界并能取到最大最小值。
- 方法论的使用: 教材指出闭区间上连续是充分条件，并用开区间/有间断反例说明边界。
- 结论: 缺少闭区间或连续性，结论可能失败。
- 结果: 使用定理前必须先检查前提。

## A2 - 触发场景

### 用户会在什么情境下需要这个 skill?

1. 证明方程在某区间内至少有一个根。
2. 证明连续函数能取到某个中间值。
3. 判断闭区间连续函数是否有最大值、最小值或有界。
4. 证明闭区间连续函数一致连续。

### 语言信号

- “零点定理”
- “介值定理”
- “闭区间连续”
- “至少有一个根”
- “最大值最小值定理”
- “一致连续”

### 与相邻 skill 的区分

- 与 `continuity-discontinuity-diagnostics` 的区别: 本 skill 使用全区间连续性定理；后者判断某个点是否连续。
- 与 `limit-existence-criteria` 的区别: 本 skill 处理函数值存在性和方程根；后者处理极限存在性。

## E - 可执行步骤

1. **核查定理前提**
   - 完成标准: 明确区间是闭区间 `[a,b]`，函数在整个闭区间连续。

2. **选择目标定理**
   - 完成标准: 有根用零点定理；取中间值用介值定理；有界/最值用最值定理；统一控制用一致连续性定理。

3. **验证附加条件**
   - 完成标准: 零点定理要端点异号；介值定理要目标值在端点函数值之间。

4. **写出存在性结论**
   - 完成标准: 明确存在点的位置，如 `c in (a,b)` 或 `c in [a,b]`，并写出满足的等式。

## B - 边界

### 不要在以下情况使用

- 区间不是闭区间，且题目没有补充端点连续性。
- 函数在区间内有间断点。
- 用户只是在求局部极限或间断点类型。

### 失败模式

- 忘记检查闭区间。
- 端点同号却套零点定理。
- 把“至少存在”误写成“唯一存在”。
