# Continuity Discontinuity Diagnostics

## R - 原文

> 如果 lim f(x)=f(x0)，那么称函数在点 x0 连续。
>
> - 同济大学数学系，第一章第八节

## I - 方法论骨架

连续性诊断是一张三项检查表。
第一，函数在该点是否有定义。
第二，该点极限是否存在；分段函数和绝对值类函数常要分别算左右极限。
第三，极限是否等于函数值。
若极限存在但函数值缺失或不匹配，是可去间断点；若左右极限都存在但不相等，是跳跃间断点；若至少一侧无穷或振荡，归入第二类间断点。

## A1 - 书中的应用

### 案例 1: 可去间断点

- 问题: 函数在某点无定义或函数值不等于极限。
- 方法论的使用: 教材通过补充或改变该点函数值，使函数连续。
- 结论: 可去间断点的本质是“极限存在，但点值问题可修复”。
- 结果: 连续性可通过补定义恢复。

### 案例 2: 跳跃间断点

- 问题: 左右极限都存在但不相等。
- 方法论的使用: 教材用分段函数示范左右极限比较。
- 结论: 总极限不存在，因此该点不连续。
- 结果: 该点是跳跃间断点。

## A2 - 触发场景

### 用户会在什么情境下需要这个 skill?

1. 分段函数在分界点处是否连续。
2. 给定函数求所有间断点并分类。
3. 要通过选择参数使函数连续。
4. 要补充或修改某点函数值使函数连续。

### 语言信号

- “是否连续”
- “间断点属于哪一类”
- “可去间断点”
- “左右极限”
- “选择 a 使函数连续”

### 与相邻 skill 的区分

- 与 `function-mapping-diagnostics` 的区别: 本 skill 假定函数对象基本合法，关注点处连续性。
- 与 `closed-interval-continuity-toolkit` 的区别: 本 skill 是局部诊断；后者用全区间连续性推出存在性。

## E - 可执行步骤

1. **定位可疑点**
   - 完成标准: 找出分母为 0、分段边界、根式/对数边界、取整跳点等位置。

2. **检查函数值**
   - 完成标准: 判断 `f(x0)` 是否定义，并记录其值。

3. **计算左右极限或总极限**
   - 完成标准: 写出左极限、右极限；若相等，再与函数值比较。

4. **分类并给出修复**
   - 完成标准: 标为连续、可去、跳跃、无穷或振荡间断；可去间断给出补定义。

## B - 边界

### 不要在以下情况使用

- 用户要利用连续性证明方程有根或取到某值。
- 函数尚未确定定义域，需先调用函数合法性诊断。

### 失败模式

- 只算右极限就断言极限存在。
- 忘记与函数值比较。
- 把可去间断和跳跃间断混淆。
