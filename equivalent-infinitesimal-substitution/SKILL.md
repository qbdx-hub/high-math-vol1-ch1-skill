# Equivalent Infinitesimal Substitution

## R - 原文

> 求两个无穷小之比的极限时，分子及分母都可用等价无穷小来代替。
>
> - 同济大学数学系，第一章第七节

## I - 方法论骨架

等价无穷小替换是局部线性化工具。
当分子、分母或乘积因子都是无穷小时，如果能找到比值趋于 1 的简单替代，就可以把复杂极限化简。
常见基础式包括 `sin x ~ x`、`tan x ~ x`、`ln(1+x) ~ x`、`e^x-1 ~ x`、`(1+x)^a-1 ~ ax`。
关键边界是：优先在乘除结构中替换；加减结构先分析主项，不能机械替换导致抵消信息丢失。

## A1 - 书中的应用

### 案例 1: `tan 2x / sin 5x`

- 问题: 三角函数比值极限。
- 方法论的使用: 教材用 `tan 2x ~ 2x`、`sin 5x ~ 5x`。
- 结论: 原极限化为 `2x/5x`。
- 结果: 极限迅速得到 `2/5`。

### 案例 2: 对数、指数和幂函数的常用等价式

- 问题: 处理 `ln(1+x)`、`e^x-1`、`(1+x)^a-1`。
- 方法论的使用: 教材由连续性、重要极限和复合极限推出这些等价关系。
- 结论: 多类初等函数在 0 附近可用首项近似。
- 结果: 第一章范围内大量未定式可统一化简。

## A2 - 触发场景

### 用户会在什么情境下需要这个 skill?

1. 求 `x->0` 时含三角、对数、指数、幂式的比值极限。
2. 用户明确说想用等价无穷小。
3. 表达式主要是乘法或除法结构。
4. 需要判断两个无穷小同阶、等价、高阶或低阶。

### 语言信号

- “等价无穷小”
- “sin x 可以换成 x 吗”
- “tan 2x / sin 5x”
- “ln(1+x)”
- “e^x-1”

- 与 `infinitesimal-limit-conversion` 的区别: 本 skill 是具体替换计算。

## E - 可执行步骤

1. **确认所有替换对象是同一过程下的无穷小**
   - 完成标准: 明确如 `x->0`，并确认被替换项趋于 0。

2. **识别结构**
   - 完成标准: 判断是乘除结构还是加减结构；加减结构先提主项或变形。

3. **选择等价式**
   - 完成标准: 写出每个替换，如 `sin(kx) ~ kx`、`ln(1+u) ~ u`。

4. **化简并检查**
   - 完成标准: 化简后的极限存在；若出现主项抵消，回到原式重新分析。

## B - 边界

### 不要在以下情况使用

- 表达式不是无穷小之比或乘除结构。
- 加减结构中存在主项抵消，未做进一步展开。
- 需要二阶以上近似，第一章未提供泰勒工具。

### 失败模式

- 在 `sin x - x` 这类差式中直接把 `sin x` 替换为 `x` 得到 0。
- 不检查 `ln(1+u)` 中 `u->0` 且 `1+u>0`。
- 混用不同趋近过程。
