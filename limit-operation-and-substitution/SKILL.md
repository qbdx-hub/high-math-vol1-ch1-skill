# Limit Operation And Substitution

## R - 原文

> 求复合函数的极限时，函数符号 f 与极限号 lim 可以交换次序。
>
> - 同济大学数学系，第一章第九节

## I - 方法论骨架

计算极限要先选路径。
如果函数在趋近点连续且点在定义区间内，优先直接代入。
如果是商、积、和、差，先检查各部分极限存在和分母极限非零，再用运算法则。
如果直接代入出现 `0/0`、`infty/infty` 等未定形式，先做代数变形、约分、有理化或变量代换。
如果出现 `sin x / x`、`(1+1/n)^n`、`(1+x)^(1/x)` 的结构，转向两个重要极限。

## A1 - 书中的应用

### 案例 1: 有理函数极限

- 问题: 分母在趋近点不为零时如何求极限。
- 方法论的使用: 教材用多项式和有理分式函数的连续性直接代入。
- 结论: 连续点处极限等于函数值。
- 结果: 直接代入是有条件的快速路径。

### 案例 2: 复合函数极限

- 问题: `f(g(x))` 的极限如何处理。
- 方法论的使用: 教材说明在内层极限存在且外层在对应点连续时，可以把极限推进函数。
- 结论: 先求内层极限，再代入外层连续函数。
- 结果: 复杂表达式可通过代换简化。

## A2 - 触发场景

### 用户会在什么情境下需要这个 skill?

1. 普通计算型极限题，没有要求严格定义证明。
2. 不知道能不能直接代入。
3. 直接代入得到未定式，需要选择变形方式。
4. 表达式中有复合函数或可换元结构。

### 语言信号

- “这个极限怎么求”
- “为什么可以直接代入”
- “这里为什么换元”
- “这个 0/0 怎么处理”
- “能不能用两个重要极限”

### 与相邻 skill 的区分

- 与 `equivalent-infinitesimal-substitution` 的区别: 如果题目主要是无穷小比值替换，用后者。
- 与 `limit-existence-criteria` 的区别: 如果目标是证明极限存在而非算值，用后者。

## E - 可执行步骤

1. **识别趋近过程和表达式类型**
   - 完成标准: 写出自变量趋向哪里，以及表达式是和差积商、复合、幂指型还是含根式。

2. **先尝试连续代入**
   - 完成标准: 检查函数在该点是否有定义且连续；若分母极限为 0，停止直接代入。

3. **处理未定式**
   - 完成标准: 选择因式分解、约分、有理化、同除最高次、变量代换或两个重要极限。

4. **回代并给出条件**
   - 完成标准: 写出所用法则的条件，如分母非零、外层连续、代换后趋近正确。

## B - 边界

### 不要在以下情况使用

- 用户明确要求 epsilon-delta 证明。
- 题目必须使用洛必达或泰勒，本试点不覆盖。

### 失败模式

- 分母趋于 0 时仍然直接套商法则。
- 复合极限中外层函数在目标点不连续。
- 换元后忘记更新趋近方向。
