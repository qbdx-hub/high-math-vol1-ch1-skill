# Limit Existence Criteria

## R - 原文

> 准则 II 单调有界数列必有极限。
>
> - 同济大学数学系，第一章第六节

## I - 方法论骨架

不是所有极限题都要先算值。
当目标表达式被两个更容易处理的对象夹住，且两边极限相同，用夹逼准则。
当数列递推、累加或逐步逼近，并能证明单调和有界，用单调有界准则先证明收敛。
函数极限也有类似的单调有界存在性思路。
这类题的核心是“证明存在性”，然后必要时再寻找极限值。

## A1 - 书中的应用

### 案例 1: 用夹逼准则证明第一个重要极限

- 问题: 证明 `sin x / x -> 1`。
- 方法论的使用: 教材用几何面积关系构造两侧界限，再让两侧同趋于 1。
- 结论: 被夹对象也趋于 1。
- 结果: 得到第一章最常用的重要极限之一。

### 案例 2: 用单调有界数列引入 e

- 问题: 证明 `(1+1/n)^n` 的极限存在。
- 方法论的使用: 教材利用单调有界数列必有极限。
- 结论: 该极限存在，并被定义为常数 e。
- 结果: 指数、对数与后续极限计算获得基础常数。

## A2 - 触发场景

### 用户会在什么情境下需要这个 skill?

1. 题目要求证明某个数列收敛，但极限值不易直接求。
2. 表达式可被上下界控制。
3. 递推数列、嵌套根式、面积逼近等结构显示单调或有界。
4. 题目出现“至少存在”“极限存在”而非“计算极限”。

### 语言信号

- “用夹逼准则”
- “单调有界”
- “证明极限存在”
- “这个数列收敛吗”
- “不求值，只证明存在”

### 与相邻 skill 的区分

- 与 `limit-operation-and-substitution` 的区别: 本 skill 先证存在；后者偏计算。
- 与 `closed-interval-continuity-toolkit` 的区别: 本 skill 处理极限存在；后者处理闭区间连续函数的存在性结论。

## E - 可执行步骤

1. **判断适合哪种准则**
   - 完成标准: 若能找到上下界且两边同极限，选夹逼；若是数列且有单调/有界结构，选单调有界。

2. **建立关键不等式**
   - 完成标准: 夹逼题写出 `g(x)<=f(x)<=h(x)`；单调有界题写出单调方向与界。

3. **验证准则条件**
   - 完成标准: 两侧极限相同，或数列同时单调且有界。

4. **给出存在性结论**
   - 完成标准: 明确写出极限存在；若可进一步算值，再转相应计算 skill。

## B - 边界

### 不要在以下情况使用

- 极限可以直接代入或简单化简得出。
- 只有有界但没有单调，不能套单调有界准则。

### 失败模式

- 上下界没有同一极限，却强行夹逼。
- 只证明有界，不证明单调。
- 把充分条件误当必要条件。
